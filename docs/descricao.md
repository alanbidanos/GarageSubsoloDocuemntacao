# 2. Descrição
O projeto é sobre uma oficina de bicicletas moderna, visando entregar o melhor serviço possível ao cliente.


## 2.1 Visão Geral do Projeto
O objetivo deste documento é fornecer uma visão geral do projeto Garage Subsolo - Oficina de Bicicletas. Ele descreve a finalidade do projeto, os principais stakeholders envolvidos, os requisitos principais, os diagramas de caso de uso, diagrama de classes e protótipos de possíveis telas para o sistema.

## 2.2 Stakeholders
- Usuários de bicicletas - Pessoas com uma pretensão a consumir produtos do ciclismo.

- Empresas de eventos de ciclismo - Empresas que cuidam de eventos relacionados ao ciclismo(triathlon, maratonas, IronMan, etc…).

- Companhias de turismo - Empresas turísticas que oferecem atrações relacionadas com ciclismo.

- Empresas de aluguel de bicicletas - Empresas que alugam bicicletas podem se interessar pelos serviços de reparos.
